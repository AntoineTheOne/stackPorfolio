Présentation du projet :

But du projet :

Le but de ce projet était de réaliser une animation 3D qui serait combinée avec les projets des autres élèves de la classe afin de créer un effet de boucle lorsqu’on passe la porte à la fin de notre projet menant vers une autre dimension (un autre projet).

Principaux défis :

  1. Animer le tapis roulant mais incliné avec la rotation de la carte

  2. Gérer l'ambiance du projet avec les sons et les effets de lumières dans le projet.

Galerie d’images:

Logiciel utilisé :

Pour ce projet, j’ai utilisé le logiciel Blender afin de concevoir et d’animer les objets 3D, puis After Effects pour combiner les rendus avec la trame sonore.

Crédits du projet :

Le projet a été entièrement réalisé par moi. Tous les objets 3D ont été conçus par mes soins, et certaines textures ainsi que certaines animations ont été créées à l’aide de tutoriels disponibles sur YouTube.
