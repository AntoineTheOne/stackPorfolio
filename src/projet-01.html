La course des étoiles

Projet scolaire / programmation

Présentation du projet :

But du projet :

Le but de ce projet était de créer un jeu de course sur un terrain à concevoir et de programmer le déplacement du véhicule afin de réaliser la course.

Principaux défis :

  1. Programmer les mouvements de la voiture de course.

  2. Générer le terrain, une tâche complex étant donné mon apprentissage récent de Unity.

Galerie d’images:

Logiciel utilisé :

Pour ce projet, j’ai utilisé Unity pour le développement du jeu et Visual Studio Code pour la programmation des scripts.

Crédits du projet :

Le projet a été entièrement programmé par moi. Le terrain ainsi que les mécaniques de jeu ont également été réalisés par mes soins, tandis que les prefabs utilisés proviennent de la bibliothèque Unity Asset Store.
